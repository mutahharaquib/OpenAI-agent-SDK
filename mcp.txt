============================================================
1. WHAT IS MCP?
============================================================

MCP = Model Context Protocol

MCP is an open standard that allows AI models to communicate with:
    -> external tools
    -> databases
    -> APIs
    -> applications
    -> local files
    -> cloud services
    -> workflows

Simple Idea:
-------------
Instead of building custom integrations for every AI app,
MCP provides ONE STANDARD METHOD.

Think of it like:
    USB-C for AI applications

Example:
---------
Without MCP:
    ChatGPT <-> Custom GitHub Integration
    Claude <-> Custom GitHub Integration
    Gemini <-> Custom GitHub Integration

Every AI app needs separate integration.

With MCP:
    AI App <-> MCP Standard <-> GitHub MCP Server

One protocol works everywhere.

============================================================

Problem Before MCP:
--------------------
N Models x M Tools problem

Example:

With MCP:

Benefits:

-> easier AI integration
-> lower engineering cost
-> portable tools
-> modular architecture
-> interoperability

3. CORE ARCHITECTURE
============================================================

Main Components:
----------------

+--------------------+
|      AI HOST       |
| (ChatGPT/Agent)    |
+----------+---------+
           |
           | MCP Protocol
           |
+----------v---------+
|     MCP CLIENT     |
+----------+---------+
           |
           | JSON-RPC
           |
+----------v---------+
|     MCP SERVER     |
+----------+---------+
           |
    +------+------+------+
    |      |      |      |
    v      v      v      v
 Database  APIs  Files  Tools

Explanation:
-------------
Host:
    The AI application

Client:
    Sends MCP requests

Server:
    Exposes tools/resources

4. HOW MCP WORKS
============================================================

Step-by-Step:
--------------

User asks:
    "Find latest GitHub issues"

AI Agent:
    Decides GitHub tool is needed

MCP Client:
    Sends request to GitHub MCP Server

GitHub MCP Server:
    Calls GitHub API

Server returns result

AI generates final answer

20. BASIC PYTHON EXAMPLES
============================================================

Example Agent:
---------------

from agents import Agent

agent = Agent(
    name="Assistant",
    instructions="Helpful AI assistant"
)

------------------------------------------------------------

Example MCP Integration:
-------------------------

from agents.mcp import MCPServerStdio

server = MCPServerStdio(
    command=["python", "server.py"]
)

------------------------------------------------------------

Example Running Agent:
-----------------------

from agents import Runner

result = Runner.run_sync(
    agent,
    "Explain MCP"
)

print(result.final_output)
